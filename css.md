## Tablas

Agregar bordes a una tabla:  

```css
table, th, td
{
    border: 1px solid black;
}
```

Uniendo los bordes de la tabla:  

```css
table, th, td
{
    border: 1px solid black;
    border-collapse: collapse;
}
```

Agregar espacio entre las celdas:

```css
th, td
{
    padding: 15px;
}
```
